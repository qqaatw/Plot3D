# Plot3D - 3D Plot Generator
![Python application](https://github.com/qqaatw/Plot3D/workflows/Python%20application/badge.svg)
## Prerequisite
 - Python 3.7.x
 - Python Packages:
   - flask
   - numpy
   - matplotlib

## Run
    python plot3DHandler.py
    
Visit http://localhost/plot3D.html to check it out.
